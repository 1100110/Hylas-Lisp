About

Hylas is a statically-typed, wide-spectrum, JIT-compiled Lisp dialect.

Features

  • Speed

    Hylas compiles to the native code of multiple platforms, and uses LLVM's aggressive optimization techniques to produce fast and efficient code. See the benchmarks.

  • True Platform-Independence

    Most languages claim to be multi-platform simply because they can compile to the machine code of many platforms - but when it comes to actually interfacing with the operating system, one has to rely on hard to test conditional compilation and platform-dependent interfaces. Hylas' standard library abstracts away any platform specific details into a single, global interface.
